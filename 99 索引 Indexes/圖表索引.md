# 📊 圖表索引

對接本書最大特色：**用圖表解構文法**。把你重繪／截取的關鍵圖表集中登錄，之後「看到圖就想起觀念」。

## 怎麼用
1. 在主題筆記的 `📊 圖表解構` 區塊重繪圖表（Markdown 表格或 Excalidraw）。
2. 圖檔存到該章的 `images/`，用 `![[檔名.png]]` 嵌入。
3. 幫該筆記加 `#圖表` 標籤，並在下表補一列。
