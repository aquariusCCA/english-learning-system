- 整理「**書**」的索引（主題地圖 MOC、圖表索引）→ 放進**該書資料夾**。
- 整理「**你**」的索引（易錯點總表、複習排程）→ 留在頂層 `99`，跨書共用、加「書」欄位標來源。
- 範本一律共用 `00`。
