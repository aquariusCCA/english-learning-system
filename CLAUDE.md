# CLAUDE.md — English Learning System 筆記庫

英文學習筆記庫（可多套教材）。目前兩套：《LiveABC 主編學堂　圖表解構英文文法》（18 章，書本）、謝孟媛英文文法（初級／中級，自學筆記整理自 Notion）。
使用者用**繁體中文**、主要在 **VS Code** 檢視；本地 git 庫（分支 `main`）。

> [!IMPORTANT]
> **動手做／改筆記前，先讀 `00 使用說明與範本/` 這三份，並照它們的規則做：**
> - `使用說明.md`（流程、忠實度原則、相容性、檢索、索引擺放）
> - `標籤系統.md`（標籤規範——**先登記再用**）
> - `範本 Templates/`（選對應**類型**的範本；文法用 `文法主題範本.md`）

## 核心規則（摘要，細節以 `00/` 為準）
- **做筆記流程**：見 `00 使用說明與範本/使用說明.md`「✍️ 記一篇新筆記的流程」（唯一真相來源，不在此重複）。
- **多書結構**：**一套教材＝一資料夾**（「書」是統稱，可為**書本**或 **Notion 課程**等；如 `01 圖表解構英文文法/…`），內含 章節（或分冊）資料夾 + `索引/`（該書 MOC、圖表索引）（＋ `Excalidraw/` 視需要）；資料夾結構詳見 `00 使用說明與範本/使用說明.md`「🗂️ 資料夾怎麼分」。
- **忠實度**：內容忠實轉錄來源、AI 補充標「💬 AI 補充」；範本雙向彈性等細節見 `00 使用說明與範本/使用說明.md`「📖 忠實度原則」（唯一真相來源，不在此重複）。
- **跨工具相容寫法**（GitHub／Obsidian 原生；VS Code 需另裝擴充——**裝哪個、為何**，見 `00 使用說明與範本/使用說明.md`「🖥️ 用什麼工具看（相容性）」）：
  - 提示框用 **GitHub-style alerts**（`> [!NOTE]`/`[!TIP]`/`[!IMPORTANT]`/`[!WARNING]`，大寫，標題放內文粗體）；可折疊用 HTML `<details>`。**不要用 Obsidian 專屬 callout**（`[!abstract]` 等小寫）。
  - 圖表：結構圖用 **Mermaid**；手繪大圖用 **Excalidraw** 匯出 SVG 嵌入。
  - 連「根 README（總導航）」用**相對連結** `[文字](../README.md)`（bare `[[README]]` 會歧義）；帶資料夾的 `[[章名/README|…]]` 才用 wikilink。
- **標籤**：主分類依**內容類型**開 namespace（`文法/*`、日後 `單字/*`…）＋通用功能標籤；**規則（子分類、命名、先登記再用）以 `標籤系統.md` 為準**。
- **索引擺放**：「**書**」的（MOC、圖表索引）放該書 `索引/`、「**你**」的（易錯點總表、複習排程）放頂層 `99 索引 Indexes/`；細節見 `00 使用說明與範本/使用說明.md`「📌 索引擺哪的原則」（唯一真相來源）。
- **原始課本截圖不進版控**（用完可刪或放 gitignore 的暫存夾）。
- **Dataview 僅 Obsidian 有效**：索引改為手動維護，勿再加 Dataview 區塊。
