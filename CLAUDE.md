# CLAUDE.md — English Learning System 筆記庫

英文學習筆記庫，可多套教材（一套教材＝一資料夾）。目前兩套：`01`《LiveABC 主編學堂　圖表解構英文文法》（書本，18 章）、`02` 謝孟媛英文文法（Notion 自學筆記，初級／中級）。
使用者用**繁體中文**、主要在 **VS Code** 檢視；本地 git 庫（分支 `main`）。

> [!IMPORTANT]
> **規則只有一份，在 `00 使用說明與範本/`。動手做／改筆記前先讀、照做：**
> - `使用說明.md` — **唯一規則手冊**：資料夾結構與索引擺放、記筆記流程（含書本截圖／Notion 兩種來源）、忠實度原則、狀態與複習、跨工具相容寫法、檢索
> - `標籤系統.md` — 標籤規範（**先登記再用**）
> - `複習方法.md` — 間隔複習怎麼做（**主動回想 > 被動重看**）
> - `範本 Templates/` — 依**內容類型**選範本（文法 → `文法主題範本.md`）
>
> 本檔**不重複任何規則內容**；若本檔或其他檔案與 `00/` 不一致，一律以 `00/` 為準。
