# 假設語氣

> [!IMPORTANT]
> **一句話核心**
> 假設語氣＝陳述**與事實相反**的假設，關鍵是**「事實與假設時態差一級」（往過去推一級）**：與**現在**事實相反 → 用**過去式**（be 一律用 **were**）；與**過去**事實相反 → 用**過去完成式**。沒有 if 的主要子句一定要放 **would／could／should／might**。if 可省略 → 倒裝（**Were／Had／Should + 主詞**）。

## 📊 四種基本句型

| 類型 | 意義 | if 子句 | 主要子句 | 例 |
| --- | --- | --- | --- | --- |
| **現在式** | 現在／未來**不確定**的假設 | If + S + **現在式** | S + **will／shall／can／may** + 原形 | If it **rains** tomorrow, we **will cancel** the picnic. |
| **過去式** | 與**現在**事實相反 | If + S + **過去式**（be→**were**） | S + **would／could／should／might** + 原形 | If I **were** free, I **would help** you. |
| **過去完成式** | 與**過去**事實相反 | If + S + **had + p.p.** | S + would／could… + **have + p.p.** | If I **had known**…, I **would have told** you. |
| **未來式（萬一）** | 未來**機率極低**（萬一） | If + S + **should** + 原形 | S + will／would／may／might + 原形 | If it **should rain**, the game **would be put off.** |
| **未來式（絕不可能）** | 未來**絕不可能**（要是） | If + S + **were to** + 原形 | S + would／might + 原形 | If the Pacific Ocean **were to dry up**, I **would change** my mind. |

> [!TIP]
> **核心觀念**
> - **時態差一級（往過去推）**：事實用現在式、假設就退到過去式；事實是過去、假設就退到過去完成式。
> - **沒有 if 的主要子句**要放 4 個助動詞表語氣：**would（←will）／could（←can）／should（←shall）／might（←may）**。
> - if 子句 vs 直述句：`If you are free, please help me.`（直述，真有可能幫）vs `If I were free, I would help you.`（假設，暗示我現在沒空）。

---

## 🟢 假設法現在式（現在或未來不確定的假設）
> 對未來做推測，**前後都用現在式**（時態不退級，因為不是「與事實相反」，只是條件未定）。

- **If it rains tomorrow, we will cancel the picnic.**（明天不一定下雨，只是條件成立就取消）
- 對照直述：I think it **will rain** tomorrow.（純陳述未來事實，非假設）

---

## 🟡 假設法過去式（與現在事實相反）
> [!WARNING]
> **be 動詞一律用 were（不分人稱）。**

- If I **were** free, I **would help** you with your homework.（＝我現在沒空）
- If I **were** young, I **could enjoy** this party.（事實：As I am not young, I cannot enjoy the party.）
- If I **had** a lot of money, I **would buy** a yacht.
- If I **were** a bird, I **would fly** around the world.

### 這兩種句型也用假設法過去式（省略的 should + 原形）
> 動詞／形容詞帶「責任、義務、建議」語氣時，that 子句用 **(should) + 原形**，should 常省略（省略後動詞仍用原形，第三人稱**不加 s**）。

- **S + suggest／demand／insist／agree… + that + S +(should)+ 原形**
  - I suggest that everyone **(should) help** in some way.
  - I insist that she **(should) consult** the doctor.
- **It + is + necessary／important… + that + S +(should)+ 原形**
  - It is important that she **(should) be** honest.
  - 對照直述（非假設）：It's important **for her to be** honest.

---

## 🔴 假設法過去完成式（與過去事實相反）
- If I **had known** the gossip then, I **would have told** you.（事實：As I didn't know…, I wouldn't tell you.）
- We **would not have dozed off** if our teacher **had spoken** louder.

### 過去式與過去完成式**混用**（兩子句時間不同 → 各自為準）
- If I **had done** my homework yesterday, I **would go** on a date with her now.
  - 與**過去**（yesterday）相反 → 過去完成式；與**現在**（now）相反 → 過去式。

---

## 🔵 假設法未來式
> 與未來相反的假設，看**可能性**選 should 或 were to。

- **should（萬一）**＝機率極低但仍可能：`If + S + should + V, S + will／would／may／might + V`
  - If it **should rain** tomorrow, the game **would (=will) be put off.**（if 子句的 should＝「萬一」）
- **were to（要是）**＝絕不可能：`If + S + were to + V, S + would／might + V`（were 不隨主詞變、主要子句必用假設）
  - If the Pacific Ocean **were to dry up**, I **would change** my mind.（暗示：我絕不改變心意）

---

## 🔀 if 的省略 → 倒裝
> 假設句可省略 if，把 **were／had／should** 移到主詞前（**不是問句**，是假設句的另一形式）。

| 原句 | 省略 if 後（倒裝） |
| --- | --- |
| If I **were** a bird, I could fly to you. | **Were I** a bird, I could fly to you. |
| If he **had** known the truth, he would have told you. | **Had he** known the truth, he would have told you. |
| If it **should** rain, I would stay at home. | **Should it** rain, I would stay at home. |

## ✳️ 其他假設法的表現（不用 if 也不用連接詞）

### as if／as though（彷彿、好像）
> 是可當連接詞的片語；與事實相反 → 差一級。

- He talks **as if** he **knew** everything.（與現在相反 → 過去式）
- She looks **as if** she **had seen** a ghost.（與過去相反 → 過去完成式）

### I wish / if only（但願，兩者意思相近）
- I wish (that) I **knew** Annie's phone number.（與現在相反 → 過去式；事實：I'm sorry I don't know…）
- I wish (that) I **hadn't bought** such an expensive dress.（與過去相反 → 過去完成式）
- **If only** my tooth **would stop** hurting!（但願／請求語氣，加 would）／**If only** I **had taken** her advice.（與過去相反）

### It is (high／about) time + that + 假設法過去式（該是…的時候了）
- It's past midnight. It's **time** (that) the children **went** to bed.（暗示還沒去睡）
- It's **about time** (that) you **apologized** to her.
- 對照：It's time **for** children **to go** to bed.（用 for + 人 + 不定詞則不需假設）

### If it were not for… / If it had not been for…（若沒有…／要不是…）
- **If it were not for** music, he **could not live** a moment.（與現在相反）
  - = **Were it not for** music…（倒裝）= **Without** music… = **But for** music…
- **If it had not been for** the traffic jam, we **would have been** in time.（與過去相反）
  - = **Had it not been for**… = **Without**… = **But for**…

## 🔁 代替 if 子句的用法
> if 子句之外，其他連接詞、不定詞、副詞片語、分詞構句等也能表假設。

### 其他連接詞
- **suppose／supposing (that)**（假設）、**providing／provided (that)**（如果）、**as long as／so long as**（只要）、**in case**（萬一）、**unless**（除非，= if…not；**通常不用於假設語氣句**）。
  - **Suppose** you had ten million dollars, what would you like to do first?
  - I will stay here **provided** the climate agrees with me tomorrow.

### 不定詞代 if 子句
- If you **heard** him talk, you would think he knew the secret. → **To hear** him talk, you would think…（主詞相同才可省、省後不可再有 if）

### 副詞連接詞（otherwise）／介系詞片語（With）
- If I **didn't know** he was innocent, I wouldn't try to save him. → I know he is innocent; **otherwise** I wouldn't try to save him.
- If you **had** a little patience, you could get over your difficulty. → **With** a little patience, you could get over your difficulty.

### 分詞構句代 if 子句
- The same man, if he **were living** in this century, would be a hero. → The same man, **living** in this century, would be a hero.（現在分詞表主動，非進行）
- If you **were seen** from a distance, you might look handsome. → **Seen** from a distance, you might look handsome.（過去分詞表被動）

### 作主詞的名詞
- If he **were** a man of sense, he would not say such a thing. → **A man of sense** wouldn't say such a thing.

## ⚠️ 易錯點分析

> [!WARNING]
> **常見錯誤（皆為來源整理的重點）**
> - **時態差一級**：與現在相反 → 過去式；與過去相反 → 過去完成式；別用錯級。
> - 假設法過去式 **be 一律用 were**（If I／he **were**…）。
> - 沒有 if 的主要子句**必放 would／could／should／might**。
> - **if 子句中 should＝「萬一」**（未來機率極低）；**were to＝「要是」**（絕不可能）。
> - **suggest／insist／It is important + that + (should) + 原形**：should 可省，動詞仍用**原形**（第三人稱不加 s）。
> - **兩子句時間不同 → 時態各自為準**（過去完成式 + 過去式混用）。
> - **if 省略要倒裝**（Were／Had／Should + S），不是問句。
> - **It's time that + 過去式**（該做卻還沒做）；用 for + 人 + 不定詞則不必假設。
> - `If it were not for`（現在）／`If it had not been for`（過去）= Without = But for。
