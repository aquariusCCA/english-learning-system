## 甲、名詞複數（01）——寫出複數（10 題）

1. bus
2. box
3. brush
4. lady
5. key
6. wife
7. shelf
8. tomato
9. piano
10. hero

<details>
<summary>✅ 甲部解答</summary>

1. **buses**（字尾 s → +es）
2. **boxes**（字尾 x → +es）
3. **brushes**（字尾 sh → +es）
4. **ladies**（子音+y → 去 y+ies）
5. **keys**（母音+y → 直接 +s）
6. **wives**（fe → ves）
7. **shelves**（f → ves）
8. **tomatoes**（子音+o → +es）
9. **pianos**（子音+o 的**例外** → 直接 +s，同 photos）
10. **heroes**（子音+o → +es）

</details>

5. fly
6. enjoy
7. have

<details>

2. **fixes**（x → +es）

5. **flies**（子音+y → 去 y+ies）
6. **enjoys**（母音+y → 直接 +s）

8. **kisses**（s → +es）
9. **does**（o → +es）
10. **plays**（母音+y → 直接 +s）

</details>

## 丙、過去式 -ed（03）——寫出規則動詞的過去式（10 題）

1. want
2. play
3. stop
4. plan
5. love
6. dance
7. study
8. cry
9. drop
10. fix

<details>
<summary>✅ 丙部解答</summary>

1. **wanted**（一般 +ed）
2. **played**（母音+y → 直接 +ed，**不去 y**）
3. **stopped**（子音+短母音+子音 → 重複字尾+ed）
4. **planned**（重複字尾+ed）
5. **loved**（字尾 e → +d）
6. **danced**（字尾 e → +d）
7. **studied**（子音+y → 去 y+ied）
8. **cried**（子音+y → 去 y+ied）
9. **dropped**（重複字尾+ed）
10. **fixed**（一般 +ed）

</details>

## 丁、跨規則對照——同一個字，三種變化（精華）

填下表的「三單（現在）」與「過去式（-ed）」，體會**同一條字尾規則怎麼跨詞形通用**。

| 原形 | 三單（現在） | 過去式（-ed） |
| --- | --- | --- |
| study |  |  |
| carry |  |  |
| play |  |  |
| stop |  |  |
| worry |  |  |

<details>
<summary>✅ 丁部解答與重點</summary>

| 原形 | 三單 | 過去式 |
| --- | --- | --- |
| study | **studies** | **studied** |
| carry | **carries** | **carried** |
| play | **plays** | **played** |
| stop | **stops** | **stopped** |
| worry | **worries** | **worried** |

兩個關鍵記法：
- **「去 y」一條分三式**：子音+y → 三單 **-ies**（studies）／過去式 **-ied**（studied）／比較級 **-ier**（happier）。**母音+y 不去 y**（play→plays／played）。
- **重複字尾只發生在過去式**：stop → 三單只加 s（**stops**）、過去式才重複字尾（**stopped**）。別把三單也重複成 ❌ stopps。

</details>
