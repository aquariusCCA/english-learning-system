# 使用說明

這份筆記系統的目標是：**系統化複習 + 方便未來檢索**。目前核心教材是《LiveABC 主編學堂　圖表解構英文文法》，未來可擴充其他英文書籍（單字、閱讀、聽力…）。

## 🗂️ 資料夾怎麼分
```
English Learning System/
├── README.md                 ← 總導航（Home / 書櫃）
├── 00 使用說明與範本/          ← 共用：規則、標籤、範本
├── 01 圖表解構英文文法/         ← 一本書＝一資料夾
│   ├── README.md             ← 全書目錄 + 快速回查
│   ├── 索引/                 ← 「這本書」的索引：主題地圖 MOC、圖表索引
│   └── 0X 章名/               ← 每章：README + 各主題筆記 + images/
└── 99 索引 Indexes/           ← 「跨書」的全域索引：易錯點總表、複習排程
```

> 📌 **索引擺哪的原則**：整理「**書**」的索引（主題地圖、圖表索引）→ 跟著書走、放該書資料夾；整理「**你**」的索引（易錯點總表、複習排程）→ 跨書共用、留在頂層 `99`。

## ✍️ 記一篇新筆記的流程
1. 從 `範本 Templates/` 選一份**對應類型的範本**複製（目前有 **文法主題範本**；不同類型的書用各自的範本，見下方 🧩）。
2. 放進該書對應的章節資料夾，檔名用 `序號 主題`（序號對齊書中小節），例：`01 主詞＋不及物動詞.md`。
3. 填 frontmatter：`chapter`、`page`、`difficulty`、`status`、`tags`。
4. 依**範本的區塊**填內容（文法主題範本：📊 圖表解構 → 🎯 核心觀念 → 📐 用法規則 → ✏️ 例句 → ⚠️ 易錯點）。
5. 回到該章 `README.md`，把該篇加進「小節導航」表並打勾。

> 🧩 **範本依「書的類型」而定**：目前只有**文法**類（`文法主題範本`）。之後若加入單字、閱讀、聽力、寫作等不同類型的書，會在 `範本 Templates/` 新增對應範本（各有自己的區塊），流程一樣、只是換範本。

> 💡 在 Obsidian 可把 `範本 Templates` 設為 Templates 外掛的範本資料夾，一鍵套用。

## 📖 忠實度原則（書本內容 vs AI 補充）
- **內容預設忠實轉錄自書本**：概念、例句、表格、注意事項以**該書原文**為準。
- **凡書本沒有、由 AI 補充的內容**（額外易錯點、自測題、延伸提醒），小節標題一律加 **💬 AI 補充**，讓你一眼區分「書 vs AI」。
- **一句話核心**為濃縮摘要，屬對書本內容的重述。
- 要查證某段是否忠實，用該篇 frontmatter 的 `page` 回書對照。

## 🔁 複習節奏（間隔複習）
每讀完一篇，把複習日排進 [[複習排程]]，建議間隔：**第 1 天 → 3 天 → 7 天 → 14 天 → 30 天**。
- 複習時：先看 **一句話核心**（`> [!IMPORTANT]` 提示框），再做「🧠 自我測驗」，最後掃「⚠️ 易錯點」。
- 每讀完一章，去 [[易錯點總表]] 把該章重點易錯點集中檢視一次。

## 🏷️ 狀態定義（frontmatter `status`）
| 狀態 | 意義 |
| --- | --- |
| 未讀 | 尚未開始 |
| 學習中 | 已建立、正在整理內容 |
| 已複習 | 完成至少一輪間隔複習 |
| 已精熟 | 自我測驗全對、能主動舉例與辨析 |

## 🔎 未來怎麼檢索
- **靠標籤**：`文法/時式`、`文法/子句`、`易錯點`…（規則見 [[標籤系統]]）。
- **靠索引**：各書自己的**主題地圖 MOC** 與**圖表索引**（在該書 `索引/`；**名稱依書而定**，例：本書的 [[文法主題地圖 MOC]]、[[圖表索引]]）；全域的 [[易錯點總表]]（考前）、[[複習排程]]。
- **靠章節 README**：每章 README 都有「快速回查」。
- **靠檔名序號**：第一次順讀、複習時跳讀。

## 🖥️ 用什麼工具看（相容性）
筆記採「跨工具相容寫法」，在 **VS Code 預覽 / GitHub / Obsidian** 三邊都能正常顯示：
- **提示框**用 GitHub-style alerts（`> [!NOTE]`／`[!WARNING]`／`[!IMPORTANT]`／`[!TIP]`）；**可折疊**內容（如自測解答）用 HTML `<details>`。
- **VS Code**：裝 **Foam** 擴充，`[[雙向連結]]` 就能點擊、並有關係圖與反向連結（近似 Obsidian 體驗）。
- **圖表**：結構圖用 **Mermaid**（GitHub 原生；VS Code 需裝「Markdown Preview Mermaid Support」擴充；Obsidian 原生）；手繪風大圖用 **Excalidraw** 匯出 SVG 嵌入（見 `01 圖表解構英文文法/Excalidraw/`）。
- **Dataview**（`99 索引` 內的自動查詢）**僅 Obsidian 支援**；其他工具會顯示為靜態程式碼區塊，不影響閱讀。
- 想要最完整體驗，用 Obsidian「Open folder as vault」開這個資料夾即可。
